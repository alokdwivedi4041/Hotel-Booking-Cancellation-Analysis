# Hotel Booking Cancellation Analysis

This project analyzes hotel booking data to identify factors influencing reservation cancellations and provide actionable recommendations to reduce revenue loss.

## Overview
High cancellation rates impact hotel revenue and operational efficiency. This analysis uncovers key drivers behind cancellations and proposes data-driven solutions.

## Key Questions
- What factors influence hotel reservation cancellations?
- How can cancellation rates be reduced?
- How can hotels optimize pricing and promotions?

## Dataset
Hotel bookings from City Hotel and Resort Hotel (July 2015 - September 2017), including booking details, customer demographics, and cancellation status.

## Tools & Libraries
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook

## Key Findings
- **Price sensitivity**: Higher room prices lead to more cancellations
- **Seasonal patterns**: August has highest bookings; January highest cancellation rate
- **Market segments**: Online TA (46%), Groups (27%), Direct (4%)
- **Geographic concentration**: Portugal has the most cancellations

## Recommendations
1. Implement dynamic pricing with discounts during peak periods
2. Launch January marketing campaigns
3. Enhance service quality in high-cancellation markets (Portugal)
4. Strengthen direct booking channels
